Authority Separation

Validation is not resolution.Malformed or unsupported fragments are refused before entering the resolver.
Resolution is not finality.A currently resolved transaction can later become conflicting when additional valid evidence appears.
Closure is bounded.SEALED means only that the current canonical fragment set matches a declared evidence boundary.
Projection is not settlement.Only resolved transactions contribute to the displayed structural projection. No payment or posting is executed.
